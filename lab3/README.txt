Набор команд:
1. new <путь к файлу> - чтение дерева из файла
2. BFS - обход в ширину
3. DFS <s/c/r> - обход в глубину (прямой/центрированный/обратный)
6. getbinarysearch - формирование двоичного дерева поиска
7. search <key> - поиск узла с ключем key (недоступно до формирования ддт)
8. balance - формирование АВЛ - дерева
9. insert <key> - вставка узла с ключем key (недоступно до балансировки)
10. delete <key> - удаление узла с клечем key (недоступно до балансировки)

Примеры команд
new tree.txt - чтение дерева из файла tree.txt
DFS c - центрированный обход в глубину
search 15 - поиск узла с ключем 15
insert 15 - вставка узла с ключем 15
delete 15 - вставка узла с ключем 15
